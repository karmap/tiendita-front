# Next.js + Tailwind CSS

This project is using [Tailwind CSS](https://tailwindcss.com/) (v2) with Next.js.
Official Docs [Tailwind docs](https://tailwindcss.com/docs/guides/nextjs).

## How to use

Install

```bash
npm i
```

Execute

```bash
npm run dev
```
